## Symbols 

| Meaning      | Command          | Output       |
|--------------|------------------|--------------|
| Addition     | `$a+b$`          | $a+b$        |
| Square Root  | `$\sqrt{2}\$`    | $\sqrt{2}\$  |
| Square       | `$a^2$`          | $a^2$        |
| AND          | `\$land\$`       | $\land$      |
| OR           | `\$lor\$`        | $\lor$       |
| NOT          | `\$lnot\$`       | $\lnot$      |
| XOR          | `\$oplus\$`      | $\oplus$     |
| THEREFORE    | `\$therefore\$`  | $\therefore$ |
| IMPLIES      | `\$implies\$`    | $\implies$   |
| IMPLIED BY   | `\$impliedby\$`  | $\impliedby$ |
| IFF          | `\$iff\$`        | $\iff$       |
| IN           | `\$in\$`         | $\in$        |
| SQUARE ROOT  | `\$sqrt{n}\$`    | $\sqrt{n}$   |
| UNION        | `\$cup\$`        | $\cup$       |
| INTERSECTION | `\$cap\$`        | $\cap$       |
| VERTICAL BAR | `\$vert\$`       | $\vert$      |
